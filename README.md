## The Problem I'm Obsessed With

Everyone's rushing to put AI in production without asking basic questions like "what happens when someone tries to trick this thing?" The answer is usually "bad things," but we keep doing it anyway.

I'm interested in the gap between academic AI security research (cool in papers, useless in production) and what actually protects you when your AI is running in the wild getting poked by every script kiddie with a ChatGPT account.

## My Philosophy (If You Can Call It That)

Security isn't about preventing all attacks—it's about making attacks expensive enough that attackers go bother someone else. Perfect security is a myth. The goal is being "secure enough" that when everything inevitably breaks, you're not the one explaining to Congress why you stored passwords in plaintext.

For AI specifically: These systems are fundamentally gullible. They're trained to be helpful, and attackers exploit that helpfulness. Plan accordingly.
