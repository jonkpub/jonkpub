## Notes On Overlap

- `fd`, `broot`, and `yazi` all help with file and directory discovery.
- `fzf` and `sk` are both fuzzy finders.
- `bat` and `eza` are both output enhancers.
- `node`, `bun`, and `pnpm` all sit in your JavaScript workflow, but at different layers.
